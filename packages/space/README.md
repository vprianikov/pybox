# Space

Shared package used across `pybox` workspaces.

## Purpose

- Keeps common tooling in one place.
- Provides a single dependency surface for experiments and notebooks.

## Included Tooling

- `ruff` for linting and formatting
- `pre-commit` for git hooks
- `commitizen` for commit message checks
- `jupyter` for notebook workflows
- `python-dotenv` for loading environment variables

## Notes

This package is managed as part of the workspace from the repository root.
